# Project Documentation

## Objective
Analyze Gurgaon real estate data using EDA.

## Steps
1. Load dataset
2. Clean data
3. Perform EDA
4. Visualize relationships
5. Generate business insights

## Business Questions
- Costliest property
- Highest average locality price
- Highest rate/sqft
- Ready vs under construction
- RERA impact
- Area vs price
- BHK analysis
- Property type
- Premium builders
