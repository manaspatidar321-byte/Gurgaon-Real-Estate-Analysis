#  Gurgaon Real Estate Analysis

# Project Overview
This project performs Exploratory Data Analysis (EDA) on residential properties in Gurgaon to generate business insights for buyers, investors, and developers.

# Business Questions
- Costliest property
- Highest average price locality
- Highest price per square foot
- Ready-to-move vs Under-construction
- RERA approval impact
- Area vs Price
- Most expensive BHK
- Costliest property type
- Premium builders

##  Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Visualizations
- Area vs Price
- Area vs Rate per Sqft

##  Key Findings
- Sector 42 has the highest rate per square foot.
- Baliawas has the highest average property price.
- Villas are the most expensive property type.
- Ready-to-move properties generally cost more.
- RERA approval does not significantly affect pricing.
